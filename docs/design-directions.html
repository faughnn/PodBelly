Direction 1
The Nook
Like settling into your favorite reading spot. Large, rounded shapes invite you in. The "now playing" card takes center stage — it's the warm hearth of the experience. Generous spacing lets everything breathe.
Fraunces serif Extra-rounded corners Warm earth tones Generous spacing
Key design moves:
• Personal greeting replaces generic "Home"
• Now-playing card is hero-sized, not a mini-player
• Episode list is compact rows, not full cards
• Duration badges instead of full metadata rows
• Dot indicator on active tab (not pill)
Direction 2
Jukebox
Energetic and alive. The layout uses multiple card sizes and a horizontal scroll to create visual variety. Numbered lists, colorful tags, and gradient progress bars add personality. It feels like a music app that's actually fun to browse.
Outfit sans-serif Gradient accents Horizontal cards Color-coded tags
Key design moves:
• Brand logo with colored accent letter
• Live pulse indicator on now-playing
• Tilted artwork (playful rotation)
• Horizontal card carousel for recently added
• Numbered list with color-coded duration tags
• Bottom tab uses underline bar, not pill

Direction 3
Cinema
The content IS the interface. A massive hero artwork dominates the screen with a frosted glass play button floating over it. Everything else recedes into darkness. Artwork glows cast subtle colored light. It feels like watching a film, not using an app.
Playfair Display serif Minimal chrome Artwork glow Ultra-dark surfaces
Key design moves:
• Hero artwork takes 40% of screen height
• Frosted glass play button overlay
• Episode info overlays gradient on artwork
• Episode list uses hairline dividers, no cards
• Each artwork casts a colored shadow glow
• Thin, lightweight bottom navigation
Direction 4
Editorial
The anti-app. Inspired by magazine and newspaper layouts. No cards, no rounded corners. Hairline dividers, precise typography, and generous whitespace. The reading experience is paramount. A single gold accent marks what's playing. It looks like nothing else in your app drawer.
Lora serif No cards Hairline dividers Light background
Key design moves:
• Light mode default (unusual for podcast apps)
• Zero rounded corners — all sharp edges
• Typography hierarchy does all the work
• Timestamps replace artwork in the episode list
• Single gold accent color, everything else is grayscale
• Text-only bottom navigation with underline
